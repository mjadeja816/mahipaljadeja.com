Google Drive File Picker Chrome Extension

This page exists to support OAuth verification and provide a transparent description of what the extension does, why Google Drive permissions are requested, and where users can read the privacy policy.

Chrome Web Store listing Extension privacy policy

What the extension does

  • Authenticates the user with Google only when the user clicks Authenticate
  • Lists Google Drive folders and files for browsing and selection
  • Downloads or exports only the user selected file to enable attachment through supported upload interfaces

Why Google Drive access is needed

  • To list files and folders so the user can choose a file
  • To download the chosen file, or export Google Docs formats to standard formats such as PDF or DOCX
  • Scope is read only and used only for the described functionality

Permissions and scope

  • Google OAuth scope: Google Drive read only
  • No write, delete, or modify operations are performed on user Drive content
  • No third party analytics or tracking is used

Policy and support

  • Privacy policy on this domain: extension-privacy.html
  • Canonical policy repository: drive-file-picker-privacy
