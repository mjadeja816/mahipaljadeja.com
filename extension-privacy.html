Privacy Policy for Google Drive File Picker Extension

This privacy policy applies to the Chrome extension listed on the Chrome Web Store as Google Drive File Picker.

Chrome Web Store listing Canonical policy repository

Data collection

  • The extension does not sell personal data
  • The extension does not use third party analytics or advertising trackers
  • The extension does not transmit Drive file contents to external servers as part of its normal operation

Google Drive access

  • Drive access is requested only after the user clicks Authenticate
  • Scope requested is Google Drive read only
  • Drive access is used to list files and to download or export only the file selected by the user

How data is used

  • To display Drive folders and files for selection
  • To download or export the user selected file for attachment through supported upload interfaces
  • To maintain basic functionality such as authentication state required by the extension

Contact

For privacy questions or support, contact: mahipaljadeja.cse@mnit.ac.in
