Privacy Policy

This Privacy Policy describes how the Chrome extension Google Drive File Picker (ChatGPT Compatible) accesses, uses, and protects Google user data.

Application Overview

The extension enables users to browse their own Google Drive, select files, and attach them to supported file upload interfaces such as ChatGPT. All actions are initiated explicitly by the user.

Data Accessed

The extension may access the following Google user data:

  • Google Drive file metadata (file name, file type, file size, modification time)
  • File content of a selected file, only when the user explicitly chooses to attach it

Data Usage

Google user data is used strictly to:

  • Display the user’s Google Drive file list
  • Allow folder navigation and file search
  • Temporarily download a user selected file
  • Attach the selected file to a supported upload interface

Data Storage

  • No Google user data is stored on external servers
  • All processing occurs locally within the user’s browser
  • No files, metadata, or OAuth tokens are persisted after the session

Data Sharing

  • No Google user data is shared with third parties
  • No data is sold, rented, or monetized
  • No analytics, tracking, or profiling is performed

OAuth & Authentication

The extension uses Google OAuth 2.0 via the Chrome Identity API. OAuth tokens are managed securely by Chrome and are never stored, logged, or transmitted by the developer.

Advertising & AI Usage

  • No Google user data is used for advertising purposes
  • No Google user data is used to train AI or machine learning models

User Control

  • Users may revoke Google Drive access at any time from their Google Account settings
  • Uninstalling the extension immediately stops all data access
