Privacy Policy

This page provides the privacy policy for the Chrome extension listed on the Tools page and related tooling. For the canonical privacy policy repository, see the GitHub link below.

Open privacy policy repository

Summary

  • No sale of personal data
  • No analytics or third party tracking
  • Permissions are used only to provide the described functionality

Data handling

  • The extension uses Google authentication only when the user clicks Authenticate
  • Google Drive access is read only and used to list and download files chosen by the user
  • No Drive files are stored on external servers by the extension

What the extension does

The extension helps the user browse Google Drive, download a selected file, and attach it through supported file upload interfaces. The user explicitly selects the file to attach.

Canonical policy

The authoritative and most up to date privacy policy is maintained here: drive-file-picker-privacy.
